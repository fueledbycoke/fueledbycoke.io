Let's make a website!

I want you to make a website following a brief that we can either let chatGPT decide or you can create something based on your design work yesterday.

The website should have multiple pages and include as many of the following things as possible:

  • Some images
  • A navigation bar that can link to other pages on your site
  • A link that anchors to a certain point on the page
  • A button that opens an alert box
  • A nicer font than this
  • Something Animated
  • A button that changes something on the page
  • A responsive page (a page that looks good on both mobile and desktop screens)

Try to think about making the layout of the pages clear and interesting.

w3schools is a great resource for HTML, CSS and Javacript.
